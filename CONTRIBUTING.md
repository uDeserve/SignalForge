# Contributing

## Scope

SignalForge is being designed as a GitHub-native feedback triage layer.

Contributions should preserve these constraints:

- `FeedbackCase` is the primary object
- GitHub is a downstream collaboration surface
- privacy is a first-class requirement
- agent delegation is optional, not mandatory

## First Changes To Make

- improve documentation
- refine object models
- tighten API contracts
- add repo skeleton and integration seams

## Review Standard

Changes should be:

- small
- explicit
- documented
- safe by default
